Python-实现常用的算法与数据结构 -- 排序

排序是一个非常经典的问题，它以一定的顺序对一个数组（或一个列表）中的项进行重新排序（可以进行比较，例如整数，浮点数，字符串等）（增加，非递减，递减， 增加，词典等）。

基于比较的三种排序算法

选择排序

  • desc

    首先，找到序列中最小的那个元素；其次，将它和序列的第一个元素交换位置（如果序列的第一个元素就是最小元素，那么它就和自己交换）；再次，在剩下的元素中找到最小的元素，将它与序列的第二个元素交换···如此往复，直到将整个序列排序。此种排序始终在不断的选择剩余元素的最小者。

插入排序

  • desc

    对于基于数组的序列，从序列的第二个元素开始（一个元素无所谓有序还是无序），和该元素左边的元素去比较，将其插入到合适的位置，然后对第三个、第四个···元素做相同的操作，直到序列的最后一个元素。也就是说，待插元素的左边是已经排好序了的，每次插入操作，都得将左边部分重新排序。

  • 实现

冒泡排序

  • desc

    给定一个N个元素的数组，冒泡法排序将：

    1. 如果元素大小关系不正确，交换这两个数（a > b）
    2. 比较一对相邻元素（a，b）
    3. 重复步骤1和2，直到我们到达数组的末尾（最后一对是第（N-2）和（N-1）项，因为我们的数组从零开始）
    4. 到目前为止，最大的元素将在最后的位置。 然后我们将N减少1，并重复步骤1，直到N = 1

稍复杂的排序算法

堆排序

  • desc

    这个其实就是利用堆这一数据结构的特点（大顶堆、小顶堆），来实现升降序的排序。Python的heapq可以快速实现堆排序。

  • 实现

归并排序

  • desc

    归并并排序是分而治之的排序算法。划分步骤很简单：

    1. 将当前数组分成两半（如果N是偶数，则将其完全平等，或者如果N是奇数，则一边稍大于一个元素）
    2. 然后递归地对这两半进行排序。
    3. 解决步骤是最有效的工作：合并两个（排序的）半部分以形成一个有序数组。

快速排序

  • desc

    分治法（divide and conquer），快排三步走：

    1. Partition：选择基准分割数组为两个子数组，小于基准和大于基准的
    2. 对两个子数组分别快排
    3. 合并结果
