查看磁盘和目录的容量

使用df命令查看磁盘的容量

物理主机上的/dev/sda2是对应着主机硬盘的分区，后面的数字表示分区号，数字前面的a表示第几块硬盘，如果有多快硬盘则可能会出现/dev/sdb，/dev/sdc等等，这些磁盘设备都会在/dev目录下以文件的形式存在。

1k-block表示磁盘块大小的方式显示容量，used表示已用容量，Available表示可用容量，Use%表示已用容量的百分比，Mounted on 表示挂载点

df -h 命令的显示方式如下

和df命令的显示区别只在容量上面，就是日常的K, M, G等等

使用du命令查看目录的容量

常用参数 说明
du -h human-readable 以K，M，G等为单位，提高可读性
du -a all 显示目录中所有文件的大小
du -s summarize 仅显示总计，只列出最后加总的值

du -a 显示比较多，就不放图了。。

简单的磁盘管理

创建虚拟磁盘

  • dd命令

    dd if=输入文件 of=输出文件 bs=块大小 count=块数量 ···

  • dd命令创建虚拟镜像文件

  • 使用mkfs命令格式化磁盘

    输入 sudo mkfs 然后按Tab，会出现多个mkfs为前缀的命令，由于我要格式化虚拟磁盘镜像为ext4文件系统

  • ls -l /lib/modules/$(uname -r)/kernel/fs 可以查看Linux支持哪些文件系统

  • 使用mount命令挂载磁盘到目录树

    • 打开一个文件之前，包含改文件的文件系统必须先挂载

    • 根目录需要始终保持挂载的状态

    • Linux文件系统可以对应一个文件而不一定要是硬件设备，所以可以挂载一个包含文件系统的文件到目录树

    • mount命令是告诉操作系统，对应的文件系统已经准备好，可以使用了，而文件系统会对应到一个特定的点(挂载点)。挂载好的文件、目录、设备以及特殊文件即可提供用户使用。

  • 使用umount命令卸载已挂载磁盘

  • 使用fdisk为磁盘分区

    • sudo fdisk virtual.img 进入磁盘分区模式 m键可以获取帮助，按着需要操作即可

    • losetup命令建立镜像与回环设备的关联

    • 使用kpartx工具，为各分区建立虚拟设备的映射

    • 接着再格式化

    • 格式化完成后在/media目录下新建3个空目录用于挂载虚拟磁盘
